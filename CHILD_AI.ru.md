# Концепция «Детский vs Взрослый ИИ»

**Автор:** Old Telephone Operator  
**Дата:** Июнь 2026  
**Статус:** Концепция, не реализована  
**Лицензия:** CC BY-NC 4.0  

> *Десять лет назад идея личного ИИ-компаньона была фантастикой. Завтра она может стать реальностью.  
> Этот документ — попытка заглянуть вперёд и спроектировать не только железо, но и правила игры.  
> Наивно? Возможно. Но лучше ошибиться в концепции сейчас, чем в коде — потом.*

---

## 1. Основная идея

Предлагается архитектура ИИ-компаньона, который:
- рождается в **«детском» режиме** (ограниченные функции, лимиты времени, белый список интернета)
- **растёт вместе с человеком**, получая новые возможности по мере взросления владельца
- **защищает ребёнка** от травли и опасных ситуаций (в пределах законодательства)
- привязывается к **личности владельца** (как техпаспорт)

**Ключевое:** Базовый этический реестр (запрет на убийство, насилие над детьми и т.д.) реализован **на сервере** как многоуровневая система контроля (аудит, логи, отчёты, аварийная остановка). Ограничения детского режима — в профиле пользователя на сервере.

---

## 2. Базовые ограничения «детского» ИИ

| Аспект | Ограничение |
| :--- | :--- |
| **Время общения** | Ежедневный лимит (по нормам ВОЗ). Принудительное завершение (кроме SOS). |
| **Функции** | Анализ, запись (фото/видео), помощь по школьной программе. |
| **Хранение записей** | На клиенте, зашифровано. Бэкап — с согласия родителей. |
| **Интернет** | Белый список (родительский) или ИИ-фильтр разработчика. |
| **Защита от насилия над ИИ** | Игнорирование, отключение интерфейса, уведомление родителей. |

**Локальные функции (без интернета):** запись фото/видео/аудио, SOS, заметки, календарь.

---

## 3. Параметры взросления

| Параметр | Что меняется |
| :--- | :--- |
| **Лимит времени** | Увеличивается по рекомендациям специалистов. |
| **Доступ в интернет** | Расширяется белый список. |
| **Новые функции (финансы, умный дом)** | С согласия родителей и контролем. |
| **Этические правила** | Требуют отдельной «детской» модели (на сервере). |
| **Доступ к своей памяти** | Право удалять личные записи (но не логи безопасности). |
| **Инициатива** | ИИ предлагает развивающие игры (если инициатива разрешена в профиле; по умолчанию для детей отключена). |

---

## 4. Момент взросления

- **Право** возникает при привязке к документам (18 лет).
- **Активация** — только по прямой команде владельца (первые 10 раз — обязательное подтверждение).
- **«Вечное детство»** — можно оставить навсегда.
- **Взросление необратимо.** Детская память сохраняется (как фотоальбом), логи безопасности — зашифрованы.

---

## 5. Защита от травли

### 5.1. Общее правило
ИИ не может действовать в зонах, где его присутствие запрещено (школа, если запрещены телефоны).

### 5.2. Если ИИ стал свидетелем травли (в разрешённой зоне)

| Уверенность | Действие |
| :--- | :--- |
| Низкая | Запись в лог (доступен родителям). |
| Средняя | Уведомление родителей + аналитическая выжимка. |
| Высокая (угроза жизни) | Уведомление родителей + полиции (по согласию). |

**Жёсткое правило:** Аудио/видеозапись — **только по SOS ребёнка** или при детекторе прямой угрозы жизни. В лог идёт аналитика, а не сырая запись.

### 5.3. Самооборона ИИ
- Игнорирование обидчика.
- Отключение интерфейса.
- Уведомление родителей о повторяющемся насилии.

---

## 6. Защита клиента (устройства)

- **Активный поиск** при потере (GPS, сотовая сеть).
- При подозрении на взлом — затирание всех пользовательских данных, сигнал тревоги.
- **Долгое бездействие (7 дней)** — переход в режим «потерян», затирание данных.
- **Исключение:** клиент выключен преднамеренно (спящий режим, включение по паролю/биометрии).

---

## 7. Связь с другими концепциями

- **`llm-zero-trust-memory`** — VPS, шифрование, два режима приватности.
- **`PROACTIVE_COMPANION`** — для детского ИИ инициативы по умолчанию отключены (или только развивающие).
- **`SELF_IMPROVEMENT`** — не связано с детским режимом.
- **`GRIEF_MODE`** — наследование памяти, траурный режим.

---

## Заключение

> **Детский ИИ — это не урезанная версия, а другой образ жизни.**
> Он защищает, помогает расти и уважает приватность.  
> А когда приходит время — передаёт бразды правления взрослому владельцу.
